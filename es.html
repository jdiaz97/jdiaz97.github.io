Clasificador de imágenes para cámaras trampas

Redes neuronales - Biodiversidad

Chile tiene 44 parques nacionales y 41 reservas naturales, casi todos estas zonas protegidas tienen cientos de cámaras trampas cada uno. Estoy construyendo una red neuronal para automatizar todo el trabajo de clasificación

Las principales limitaciones de este trabajo son: un set de datos muy limitado y muy malas fotografías, pero la IA funciona de todas formas. Incluye especies nativas y exóticas..

En el futuro planeo expandir esta red neuronal y aplicarla en conjunto con drones para monitorear el avance de especies invasoras como el castor.
